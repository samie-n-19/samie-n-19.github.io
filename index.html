ABOUT ME

I'm a Robotics & Automation graduate student at WPI, researching dexterous in-hand manipulation using vibration-induced friction modulation in the MER Lab. My work bridges hardware and software — from PCB design and embedded firmware to deep learning inference and cloud-connected backends.

Before WPI, I built production healthcare systems at Persistent Systems using Java and AWS, engineered UI features for Zendesk in Node.js + React, and interned at DRDO designing EtherCAT controller PCBs.

I thrive at the intersection of intelligent perception, physical systems, and real-world deployment.
